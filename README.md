# 🛡️ Shadow SOS – AI-Powered Silent Emergency Detection System 

## 🌐 Live Demo

🔗 https://shadow-sos.lovable.app

## Overview

Shadow SOS is an AI-powered emergency monitoring system designed to silently detect potential emergencies using smartphone sensor data. This repository contains a responsive web-based prototype that demonstrates how the system monitors user activity, analyzes motion patterns, and automatically initiates an emergency response when a dangerous situation is detected.

The website simulates the complete emergency detection workflow through an interactive dashboard, providing a realistic demonstration of the project's concept and functionality.

---

## Problem Statement

In many emergency situations, victims may be unable to unlock their phone or manually trigger an SOS alert. Existing safety applications often depend on user interaction, which may not always be possible during accidents, medical emergencies, or personal safety incidents.

Shadow SOS addresses this challenge by demonstrating how AI can automatically detect suspicious events and initiate emergency assistance without requiring manual intervention.

## Demo Workflow

1. Continuous sensor monitoring begins.
2. Motion data is analyzed by the AI engine.
3. Abnormal activity is detected.
4. Risk level increases.
5. A verification countdown starts.
6. The user can cancel the alert.
7. If no response is received, an SOS alert is triggered.
8. Live location is shared with trusted contacts.
9. The emergency event is recorded in the history log.
